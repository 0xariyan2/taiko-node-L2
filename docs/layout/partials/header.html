Reliable browser automation

Taiko is a browser automation tool built by the team behind Gauge by ThoughtWorks. It is a JavaScript library (on Node.js) built with a clear and concise API to automate the chrome browser.

Features

Tests written in Taiko are highly readable and maintainable. Taiko is built ground up to test modern web applications, and knows how to handle AJAX and dynamic content.

Taiko comes with a REPL to easily record and write tests. Taiko’s unique API and selectors allows you to write tests without peeking into web application’s source code or dealing with XPaths.

Installation

Get Started with Taiko

Install with npm globally:

        $ npm install --global taiko

or as a development dependency for your project:

          $ npm install --save-dev taiko
Note:

On Windows, make sure that the location %AppData%\npm (or wherever npm ends up installing the module on your Windows flavor) is present in PATH environment variable.

Note:

On Linux, install taiko to a [NODE_PATH] with executable permission.
